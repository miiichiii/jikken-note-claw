# ミーティング議事録サマリー：MCTO研究の不一致解消に向けて

**参加者**: 濱田教授 (Hamada Lab) vs Bjorn教授 (Harvard/Baron Lab)
**目的**: データの不一致解消、共同研究・論文投稿方針の協議

## 1. In Vivo (生体内) データ
- **一致点**: MCTOマウスは遺伝子量依存的に**低骨密度 (BV/TV低下)** を示す。
- **Bjorn側の発見 (骨形成)**: 骨形成マーカー(P1NP)・骨形成率(BFR)が著しく**上昇**。骨芽細胞活性も高い。
- **Bjorn側の発見 (骨吸収)**: 破骨細胞数は増加傾向だが、血清マーカー(TRAP5B)は**低下** → **「数は増えているが機能は低下している」**可能性。

## 2. In Vitro (培養実験) の重大な不一致
- **Bjorn側 (RankL 10ng/ml)**: 破骨細胞分化・吸収機能が**著しく抑制**される。
- **Hamada側 (RankL 50ng/ml)**: 破骨細胞分化が**促進**される。
- **仮説**: RankL濃度の違い (10 vs 50) が、Mafb発現制御を通じて結果を逆転させている可能性。

## 3. RNA-seq解析
- Bjorn側は7組織のBulk RNA-seq & scRNA-seqを進行中。
- 予備データ: 免疫関連、TGF-beta、BMP、ケモカイン経路の変化、In vivoでのRankL/MCSF増加を確認。
- **問題点**: Hamada論文の特定マーカー変化がBjorn側で再現されていない。

## 4. アクションプラン
- **再現実験**: 両ラボで同一条件 (細胞密度) ＋ **RankL 10ng/ml と 50ng/ml の両条件** で実験を行う。
- **サンプル交換**: Hamada側の骨髄移植サンプルをBjorn側に送り、詳細な骨形態計測を実施。
- **論文**: 不一致を解消後、**Back-to-back (2報同時投稿)** を目指す。
