# 入試面接用 質問リスト (2026年版)

初めての面接官として、受験生の「数値化できない適性（職業倫理・対人能力・自己管理能力）」を引き出すための質問集です。

## 1. 定番・基本質問 (ビジョンと基礎素養)
他の先生と重なる可能性がありますが、比較の軸として有効です。

*   **ビジョン**: 「本学で学ぶことで、将来どんな医療人になりたいと考えていますか？」
*   **レジリエンス**: 「これまでの人生で、一番『壁』にぶつかった経験と、それをどう乗り越えたか教えてください。」
*   **協調性**: 「チームや集団で動くとき、あなたが一番大切にしていることは何ですか？」

## 2. 深掘り・抽象的質問 (価値観と人間性)
他の先生と視点を変えて評価したい場合に有効です。

*   **倫理観・価値観**: 「あなたが考える『信頼される医療人』に最も必要な資質は何だと思いますか？」
    *   *見るポイント*: 技術重視か、誠実さ重視か、その根底にあるスタンス。
*   **多様性・寛容性**: 「自分とは全く異なる考え方や価値観を持つ人と接するとき、あなたはどのようなことを心がけますか？」
    *   *見るポイント*: 患者さんや多職種連携において必要な「心の柔軟性」。
*   **自己研鑽・謙虚さ**: 「医療の世界は日々進歩していますが、プロになった後、学び続けるために大切にしたいことは何ですか？」
    *   *見るポイント*: 国家試験合格をゴールとせず、一生学び続ける覚悟があるか。
*   **社会貢献・地域意識**: 「本学は地域に根ざした大学ですが、地域の医療を支える上で、大学や学生に求められている役割は何だと思いますか？」
    *   *見るポイント*: 公立大学の学生としての自覚と視野の広さ。
*   **自己客観視・リスク管理**: 「自分の弱点や短所を一つ挙げてください。また、それを医療現場でどのようにコントロールしようと考えていますか？」
    *   *見るポイント*: 自分の弱さがミスに繋がらないよう対策するプロ意識。

## 3. 書類（調査書等）に基づく質問例
*   **部活動**: 「部活動で意見が対立した時、あなたはどのように振る舞いましたか？」
*   **ボランティア**: 「活動を通じて、自分の中での『医療』や『福祉』のイメージはどう変わりましたか？」
*   **学習成績**: 「得意な（または苦手な）科目に対して、どのように向き合ってきましたか？」
