# 研究背景：MAFB変異によるMCTO発症メカニズムの解明

## 1. 研究の対象と目的
- **対象**: 超希少な骨吸収疾患「多中心性手根足根骨溶解症 (MCTO)」
- **原因**: 転写因子 **MAFB** の酸性転写活性化ドメイン（TAD）における変異。
- **目的**: 詳細な分子メカニズムの解明（現状は未解明）。

## 2. 使用しているモデル系
- **MCTO KIマウス**: 
    - 患者と同様の変異（Mafb MCTO変異）を持つ。
    - **表現型**: 進行性の骨溶解（骨密度低下、手根骨・足根骨の縮小）。
- **Myeloid-specific Mafb-deficientマウス (Mafb f/f :: LysM-Cre)**:
    - 骨髄系細胞でMafbを欠損。
    - **表現型**: MCTOとは対照的に、大理石骨病様の**高骨量**を示す。

## 3. 現在直面している課題（ラボ間のデータ乖離）
共同研究先（海外ラボ）と濱田ラボで、in vitro破骨細胞分化のデータが真逆になっている。

| 項目 | 濱田ラボ（本稿） | 共同研究先 |
| :--- | :--- | :--- |
| **MCTO KI破骨細胞分化** | **亢進**（TRAP陽性多核細胞数増加） | **抑制**（分化・融合が著しく阻害） |
| **コロニーアッセイ (BM)** | **MCTOで増加** (Avg 35.0 vs 12.7) | - |
| **骨髄細胞数 (Total BM)** | **MCTOで減少** (~50% of Ctrl) | - |
| **骨吸収活性 (Pit assay)** | 軽微な変化 | 著しく低下 |
| **RANKL刺激濃度** | 50 ng/mL（強刺激） | 10 ng/mL（弱刺激） |
| **表現型の解釈** | MAFBは機能成熟の「加減抵抗器」 | MCTO変異は機能喪失または抑制機能の喪失 |

## 4. 解決すべきポイント
1.  **刺激条件の依存性**: RANKL濃度・培養期間がMAFBの分解速度や標的遺伝子に与える影響（正・負の逆転メカニズム）の特定。
2.  **オミクスデータの統合解析**: Bulk/Single-cell RNA-seq再解析により、MAFBが「どのフェーズ」で「どの代謝路（リソソーム、ミトコンドリア等）」を制御しているか精密定義。
3.  **In vivo vs In vitro の乖離**: 生体（骨溶解）と試験管（分化抑制＠相手ラボ）の矛盾を解くカギ（液性因子など）の探索。
