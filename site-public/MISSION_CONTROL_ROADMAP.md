# Mission Control Roadmap (site-public Evolution)

このファイルは、現在の「研究レポートサイト」をセンセイ専用の「Mission Control（総合管理画面）」へと進化させるための実装アイデアとToDoリストです。

## 🎯 最終ゴール
研究、大学公務、私的な予定、およびAIエージェントの活動を一元的に可視化・管理できるウェブインターフェースの構築。

## 📋 実装アイデア・ToDo

### Phase 1: タスクと情報の可視化 (Current Focus)
- [x] **タスクボードの実装**:
    - 抱えているタスク（入試業務、DPチェック表、研究解析等）をカード形式で表示。
    - ステータス（未着手・進行中・完了）の管理。
- [x] **カレンダー・スケジュールの統合**:
    - Googleカレンダーの予定表示。
    - AIの定期ジョブ（メールチェック、深夜レポート）の実行予定の可視化。

### Phase 2: メモリーと検索の強化
- [ ] **メモリー・ブラウザ**:
    - `memory/` フォルダ内の過去ログを日付別・プロジェクト別に閲覧できる画面。
- [ ] **全文検索機能**:
    - 過去の対話やメモから必要な情報を即座に引き出す検索窓の設置。

### Phase 3: エージェント（チーム）の管理
- [ ] **AIチームステータス画面**:
    - 現在稼働中のサブエージェント（DeepSeek, GPT-OSS等）の活動状況をアバター付きで表示。
    - 誰がどのタスクを担当しているかの明確化。

### Phase 4: デザインとUIの洗練
- [ ] **レスポンシブ・ダッシュボード**:
    - スマホからでも一目で状況がわかるモダンなカードUIデザイン。
    - ダークモード/ライトモードの対応。
