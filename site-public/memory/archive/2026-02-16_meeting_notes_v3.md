# 2026-02-16 会議メモ（高橋先生退官記念）その3

## 1. シンポジウムと企業協賛・展示
- **企業ブース/ポスター**: 
    - シンポジウム会場（国際会議場 多目的ホールなど）の受付付近やホールの壁際を活用して、企業パンフレットの設置や簡単な説明ブースを設ける。
    - これにより「協賛費」として運営資金を得るスキームを検討。
    - 学生のポスター発表（解剖発生やセンター関連）も併設し、コーヒーブレーク時に見てもらうことで、シンポジウムとしての実体を強化する。
- **会場の広さ**: 
    - 多目的ホールは正方形で200〜300名規模の収容が可能（食事付きパーティーも対応可）との認識。詳細は3月の下見・予約時に再確認。

    - 17:00：祝賀会開始
    - 19:00：終了（早めに終わらせて、遠方の参加者や二次会へ配慮）
- **当日スタッフ**:
    - 平日の事務方には頼みにくいため、高橋研のOB/OG、または現役生に「お手伝い」として協力をお願いする。
